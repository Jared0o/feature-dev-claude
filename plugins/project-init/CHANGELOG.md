# Changelog

## 0.1.0 — initial release

- `/project-init:init` — greenfield workflow with gated phases (discovery → modules → architecture → scaffold → wire → verify → CLAUDE.md).
- `/project-init:add-module` — adds a new module to an existing solution.
- Templates modeled on [Jared0o/Ecommerce](https://github.com/Jared0o/Ecommerce): .NET 10 modular monolith, Mediator, `Result<T>`, EF Core + PostgreSQL per-module, xunit + NSubstitute + Testcontainers.
